# ProjectTwo_Team02

## Steps To Run

### 1. Install Maven
Debian Linux: `sudo apt-get install maven`\
Mac: `brew install maven`\
Windows `???`

### 2. Build Project

`mvn clean package`

### 3. Run The JAR
#### Server
`java -jar Server/target/Server-1.0-SNAPSHOT-jar-with-dependencies.jar `
#### Client
`java -jar Client/target/Client-1.0-SNAPSHOT-jar-with-dependencies.jar`
